Libre-Subs FAQ

How can I contact the dev?

FifthWit can be reached via our GitHub repository at github.com/FifthWit/Libre-Subs

Are there any restrictions?

Libre-Subs has absolutely no restrictions, you can use it for anything you want. However much you want.

Do you take donations?

Yes, we accept donations via Ko-fi. If you want to support the project, please visit ko-fi.com/fifthwit.

How to implement this into my project?

Check out our open-source repository at github.com/FifthWit/Libre-Subs for implementation details and documentation!

Can I use this for a streaming site?

Almost all of our users are streaming sites, so yes, you can definitely use it for a streaming site.
